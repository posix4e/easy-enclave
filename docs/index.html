01

Install the SDK

Get the Python client to connect to attested services.

pip install easyenclave
02

Deploy Your Service

Add our GitHub Action to your repo. Push a docker-compose and we handle the rest.

name: Deploy to TDX
on: workflow_dispatch

jobs:
  deploy:
    uses: anthropics/easyenclave/.github/workflows/pipeline-dev.yml@main
    secrets: inherit
03

Connect & Verify

Connect to your service. We verify the TDX attestation automatically.

from easyenclave import connect

# That's it. Attestation verified.
client = connect("your-org/your-repo")
response = client.get("/api/data")
